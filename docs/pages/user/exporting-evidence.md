# Exporting Evidence

The export package replaces the old SSRS PDF/Excel workflow.

## Required exports

- HTML report for browser review
- XLSX workbook for data inspection
- JSON manifest for run metadata
- CSV tables for raw result sections

## Later export

PDF export should be generated from the same HTML report used by the browser view.
