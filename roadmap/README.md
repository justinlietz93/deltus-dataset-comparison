# Roadmap TODO Waves

This folder contains phase/task/step checklists for the polish waves after the `0.1.0` MVP.

## Wave order

1. `../TODO.md` - `0.1.0` MVP: first useful SQL Server comparison evidence path.
2. `0.2.x-engine-hardening/TODO.md` - harden the SQL Server engine and legacy behavior parity.
3. `0.3.x-report-reader-exports/TODO.md` - turn raw evidence into a polished report reader and export surface.
4. `0.4.x-workflow-adapters/TODO.md` - add workflow breadth, local file adapters, and run management.
5. `0.5.x-team-polish/TODO.md` - make it team-usable, packageable, and demonstrable.
6. `1.x.x-stable-product/TODO.md` - complete the stable product line.

## Roadmap rule

Each wave must preserve the same center:

> Deltus produces trustworthy before/after evidence for heterogeneous tabular data without making the comparison engine depend on a specific UI, database, or report renderer.
